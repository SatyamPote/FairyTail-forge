PROJECT NAME: MangaForge Mobile

Build a FULLY OFFLINE AI manga/comic generation engine optimized for MOBILE DEVICES.

==================================================
CORE GOAL
=========

Create a lightweight AI manga generator that:

* works fully offline
* runs on Android and iOS
* generates ONLY black-and-white manga line art
* generates comic stories locally
* creates comic pages
* uses SMALL models
* runs FAST on mobile GPUs and CPUs

DO NOT use:

* cloud APIs
* OpenAI APIs
* Ollama
* SDXL
* Flux
* huge models
* photorealistic generation

This software must prioritize:

* speed
* low RAM
* low VRAM
* manga line art
* mobile optimization
