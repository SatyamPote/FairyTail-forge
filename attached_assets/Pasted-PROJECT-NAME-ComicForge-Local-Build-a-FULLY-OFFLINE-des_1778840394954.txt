PROJECT NAME: ComicForge Local

Build a FULLY OFFLINE desktop AI comic generation software.

CORE GOAL:
Create a local-first AI comic studio that:

* writes stories
* generates comic scripts
* creates black-and-white manga/comic line art
* exports comic pages
* works offline
* uses local AI only
* uses very small models
* runs on consumer GPUs
* has fast generation speed
* no cloud APIs
* no subscriptions
* no internet required for generation

==================================================
IMPORTANT REQUIREMENTS
======================

The image generation model MUST:

* be lightweight
* under 2GB total if possible
* optimized for black-and-white line art only
* NOT generate color images
* optimized specifically for:

  * manga line art
  * comic sketches
  * ink drawings
  * monochrome illustrations
  * cel-shaded outlines
  * storyboard visuals

The model should:

* generate FAST
* use low VRAM
* prioritize speed over photorealism
* work locally on RTX 3060+

The software MUST:

* generate stories locally
* generate images locally
* generate comic pages locally
* save projects locally
* support offline workflow

Internet access should ONLY be optional for:

* story research
* web search for references
* downloading models

IMAGE MODEL REQUIREMENTS
========================

DO NOT USE:

* heavy SDXL models
* photorealistic models
* large Flux models

USE:

* tiny SD models
* distilled models
* manga-focused models
* line-art-only models

Target:

* 512x512
* monochrome output
* 5-10 steps generation
* very fast inference

Possible approaches:

1. Tiny Stable Diffusion
2. SD-Turbo distilled
3. LCM models
4. Custom UNet reduction
5. LoRA-only line-art specialization
6. Kandinsky-lite
7. Manga diffusion fine-tune
8. ONNX optimized inference
